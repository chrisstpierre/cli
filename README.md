# Asyncy CI

The Asyncy CLI is used to manage Asyncy from the command line.

## Overview

The goals of this project is to provide a utility for developers to interact with all of Asyncy features/services.

## Installation

```shell
$ pip install --user asyncy
$ asyncy login
```
